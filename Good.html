Library Management System (Java - OOP Implementation)

A console-based library management system developed in Java using Object-Oriented Programming (OOP) principles. It demonstrates encapsulation, inheritance, abstraction, and polymorphism in managing books, magazines, and members with file persistence.

  • Implementation: Java (Object-Oriented Design with File Serialization for persistent data storage).
  • Core Classes: LibraryItem, Book, Magazine, Member, and LibraryManagementSystem.
  • Data Management: Stores and retrieves all records (Books, Magazines, Members, Borrow/Return logs) using serialized files.
  • OOP Concepts: Demonstrates Encapsulation, Inheritance, Abstraction, and Polymorphism.
  • Key Features: Admin Login, Member Registration, Borrow & Return Management, Auto-ID Generation, Quantity Updates, and Lost Item Tracking.
  • Code Reference: LibraryManagementSystem.java (View on GitHub)
